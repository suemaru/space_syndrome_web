GAME CONTENT ゲームコンテンツ

TYPE 1

カジュアルゲーム

対象: 小学生以上

パズルやシューティングなど、誰でも手軽に楽しめるゲーム。獲得ポイントでアイテムと交換可能。

TYPE 2

ビルドアップ・シミュレーション

対象: 小学生以上 / 18歳以上版

実在する地方の土地を舞台に、町や建築物を創造。成果物は取引場で売買可能。

TYPE 3

国取り合戦

対象: 高校生以上

戦略性の高いRPG×RTS。地域の拠点を巡る壮大な戦い。メインコンテンツとして展開。

ゲームの特徴

  • 実在する地方の不動産を活用したリアルな舞台設定
  • 地域特産物や歴史をゲームアイテム化
  • NFT化による資産価値の創出
  • プレイヤー間での自由な取引システム
